栅格系统

Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统， 随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。

内联表单

添加 .form-inline 类可使其内容左对齐并且表现为 inline-block 级别的控件。

焦点状态

我们将某些表单控件的默认 outline 样式移除，然后对 :focus 状态赋予 box-shadow 属性。

按钮

可作为按钮使用的标签或元素.

字体图标

包括250多个来自 Glyphicon Halflings 的字体图标。 Glyphicons Halflings 一般是收费的，但是他们的作者允许 Bootstrap 免费使用。

按钮组

通过按钮组容器把一组按钮放在同一行里。通过与按钮插件联合使用， 可以设置为单选框或多选框的样式和行为。

导航条

导航条是在您的应用或网站中作为导航页头的响应式基础组件。 它们在移动设备上可以折叠（并且可开可关）， 且在视口（viewport）宽度增加时逐渐变为水平展开模式。

缩略图

通过缩略图组件扩展 Bootstrap 的 栅格系统，可以很容易地展示栅格样式的图像、视频、文本等内容。
